Introduction

The induction principle comes from a generalization of a dependent function that makes recursion on natural numbers. We first define what is a natural number and later, we show how to define functions by means of the recursor and induction principles.

The universe will be denoted by ùí∞:

ùí∞ = Set

If we follow an algorithmic procedure to define natural numbers, we will encounter two rules to construct them, one rule to get the zero and another rule for the rest, that we call it, suc for successor.

Recursion principle

The principle of primitive recursion for natural numbers establishes that to create a non-dependent outgoing function with natural numbers as its domain, we just need to tell it the type in the codomain, and the function behavior on the ‚Ñï constructors. Which means, says what return in the case when the number is zero and when the number was built using suc constructor.

In Agda, the rec‚Ñï also called recursor for natural number has been defined by pattern-matching as follows.

  : (C : ùí∞)
  ‚Üí C
  ‚Üí (‚Ñï ‚Üí C ‚Üí C)
  -------------
  ‚Üí ‚Ñï ‚Üí C

With the following equations:

rec‚Ñï C c‚ÇÄ c‚Çõ zero    = c‚ÇÄ
rec‚Ñï C c‚ÇÄ c‚Çõ (suc n) = c‚Çõ n (rec‚Ñï C c‚ÇÄ c‚Çõ n)

Induction principle

The induction here is a generalization of the principle of recursion. In first-order we can write the induction schema or the principle of mathematical induction.

In particular, a property of natural numbers is represented by a family of types P : ‚Ñï ‚Üí ùí∞. From this point of view, the above induction principle says that if we can prove P(0), and if for any n we can prove P(succ(n)) assuming P(n), then we have P(n) for all n. (Pp. 50-51 in (Univalent Foundations Program, 2013))

By using a dependent function to obtain its version in type theory we have the following

ind‚Ñï
  : ‚àÄ {C : ‚Ñï ‚Üí ùí∞}
  ‚Üí C zero
  ‚Üí (‚àÄ (n : ‚Ñï) ‚Üí C n ‚Üí C (suc n))
  -------------------
  ‚Üí ‚àÄ (n : ‚Ñï) ‚Üí C n

with the defining equations

ind‚Ñï c‚ÇÄ c‚Çõ zero    = c‚ÇÄ
ind‚Ñï c‚ÇÄ c‚Çõ (suc n) = c‚Çõ n (ind‚Ñï c‚ÇÄ c‚Çõ n)

Now, we have the power of induction to prove some properties.

Alternative induction

Assuming the ordinary induction principle (i.e., ind‚Ñï) derives the transfinite induction principle.
For a unary predicate $$P : \mathbb{N} \to \mathcal{U}$$, if

$$ \prod\limits_{n : \mathbb{N}} \left ( \prod\limits_{k : \mathbb{N}} (k < n \to P(k)) \to P(n) \right) $$

then for all $$n : \mathbb{N}$$, $$P(n)$$.

To solve this problem, we need to define a type for the less than relationship (_<_) between natural numbers but we also have to define the disjunction to make a case analysis that we need for our proof. Let‚Äôs see. You may skip this first part.
