Leibniz's Equality

Leibniz characterised the notion of equality as follows: Given any and , if and only if, given any predicate , if and only if .

To denote the equality between two elements we use the symbol (_≡_). In Agda we will formalize such a notation of equality with Eq type.

Eq : ∀ {A : Set} → (x y : A) → Set₁
Eq {A} x y = (P : A → Set) → (P x → P y)

We can think about this equality definition saying that is equal to if and only if every property (unary predicate variable ) that satisfies, satisfies as well.

  • Reflexivity
refl : ∀ {A : Set} → (x : A) → Eq x x
refl {A} x = λ P Px₁ → Px₁
  • Transitivity
trans : ∀ {A : Set} → (x y z : A) → Eq x y → Eq y z → Eq x z
trans {A} x y z x≡y P₁ y≡z P₂ = P₁ y≡z (x≡y y≡z P₂)
  • Symmetry
sym : ∀ {A : Set} → (x y : A) → Eq x y → Eq y x
sym {A} x y x≡y P = x≡y p₁ (λ z → z)
  where
    p₁ : A → Set
    p₁ = λ z → P z → P x
